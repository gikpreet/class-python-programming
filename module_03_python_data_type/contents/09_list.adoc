= list 타입

---

* 슬라이싱을 사용하여 list를 재할당 할 수 있습니다.

[source, python]
----
l = [1,2,3,4,5,6,7,8,9]
l[2:4] = ['a', 'b']
print(l)
----

----
[1, 2, 'a', 'b', 5, 6, 7, 8, 9]
----

* 슬라이싱의 사용하여 list내의 요소를 삭제할 수 있습니다.

[source, python]
----
l = [1,2,3,4,5,6,7,8,9]
del l[2:4]
print(l)
----
